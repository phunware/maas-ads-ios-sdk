PWAdsRewardedVideoDelegate Protocol Reference

Conforms to NSObject
Declared in PWAdsRewardedVideo.h

Overview

A PWAdsRewardedVideoDelegate is needed to receive callbacks about rewarded video ad status.

Parameters
